Artefacts

Data Structures

Python's main data structures include lists, dictionaries and tuples. Lists are useful to contain multiple values used in the same context as each other, and they can easily be iterated upon and edited. Tuples do not allow editing, and are otherwise similar to lists. They are useful for containing data which should never be modified, such as state machine data. Finally, dictionaries come in key-value pairs. This makes them useful as a way to store labelled data of different types. More entries can be added during runtime. Find the dictionary exercise here.
