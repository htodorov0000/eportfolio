Information Security Management

Module 5

Expected Learning Outcomes

1. Identify and analyse security risks, threats and vulnerabilities in information systems and determine appropriate methodologies, tools, and techniques to manage and/or solve them.
2. Gather and synthesise information from multiple sources (including internet security alerts & warning sites) to aid in the systematic analysis of security breaches & issues.
3. Critically appraise and utilise methodologies, tools and techniques that help manage and audit security issues.
4. Articulate the legal, social, ethical, and professional issues faced by information security professionals.
