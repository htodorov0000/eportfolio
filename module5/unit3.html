Unit 3

Introduction to Threat Modelling and Management

Learning Outcomes

1. Describe a number of threat modelling techniques.
2. Advise which technique should be used in specific situations.
3. Discuss when techniques should be combined in a hybrid model.

Artefacts

Security standards exercise:
1. The company discussed in the assessment would benefit most from the PCI Security Standards, as they apply broadly to data security with regards to B2C retail. Assuming the company decides to proceed with entering the global supply chain, the GDPR standards would also become vital to follow.
2. With regards to the PCI standards, I would evaluate the data encryption and the security of the network which the service uses.
