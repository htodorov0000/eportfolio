Unit 3

Vulnerability Assessments

Learning Outcomes

1. Select and explain which assessment regime to use based on the type and size of the target business.
2. Utilise open-source scanning and testing tools to perform vulnerability tests and scans.
3. Analyse scanning results and make recommendations to mitigate vulnerabilities.

Artefacts

I performed the scanning activity, which required scanning using common tools to find out information about the previously selected site's network. Findings: It took 20 hops from my machine to the server, with several being timeout requests. The longer delays took about 300 ms. The website is hosted with AWS.

Reflection

Due to my little available time I mostly focused on finishing up the assignment during this unit. I found that, as usual, I learn better by doing, so I don't believe this was detrimental for my learning experience.
