Vulnerability Analysis – Literature Review Activity

According to OWASP (2024), the following are some of the common generic website vulnerabilities:

  • Injection
  • Broken Access Control
  • Cryptographic failures
  • Insecure Design
  • Vulnerable and Outdated Components
Furthermore, the following vulnerabilities are expected to be relevant and exploitable at a first glance:
  • Outdated protocol (HTTP)
  • SQL Injection
  • Cross-Site Scripting (XSS)
References:
OWASP (2024) OWASP Top Ten. Available at: https://owasp.org/www-project-top-ten/.

Reflection

The task of this unit, which is also relevant for the first assignment, allowed me to learn more about different vulnerabilities and begin testing them against the chosen website. This was valuable in understanding the basics of penetration testing.
